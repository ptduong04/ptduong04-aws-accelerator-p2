**Date:** 05/06/2026 (Onsite)

## Learning Objectives
- Implement horizontal and vertical scaling
- Configure advanced networking
- Deploy applications on minikube
- Understand service mesh basics

---

## Topics Covered

### 1. Scaling Strategies

#### Horizontal Pod Autoscaler (HPA)
- Scale based on CPU/memory metrics
- Custom metrics
- Min/max replicas

#### Vertical Pod Autoscaler (VPA)
- Adjust resource requests/limits
- Recommendation mode vs auto mode

### 2. Advanced Networking

#### Ingress
- HTTP/HTTPS routing
- Path-based routing
- Host-based routing

#### Service Mesh
- Traffic management
- Observability
- Security

### 3. Storage

#### Volumes
- EmptyDir, HostPath
- PersistentVolume (PV)
- PersistentVolumeClaim (PVC)

#### StatefulSets
- Stable network identities
- Ordered deployment/scaling
- Persistent storage per pod
